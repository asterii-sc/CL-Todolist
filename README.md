Python CLI To-Do List
A simple command-line interface for managing tasks, built with Python. This tool uses a local JSON file to ensure your data persists between sessions.
⚙️ Features
Persistent Storage: Tasks are saved to todos.json.
Reliable Paths: Uses os.path to locate data files regardless of where the script is executed.
Error Handling: Gracefully handles missing JSON files and invalid user input.

USAGE:
1. ensure you have the latest version of python installed.
2. download the passcode.py and .json file into the same folder
3. run the script
